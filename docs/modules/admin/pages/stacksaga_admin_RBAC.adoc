== Role-based access control (RBAC)

.ROLES
|===
|ROLE |Permissions |Type

|SUPER_ADMIN
|- Manage Admin Users
|USER_TYPE

|ADMIN
|- Manage OPERATOR_ROLE +
- Manage GATEWAY_ROLE +
- Manage SERVICE_ROLE
|USER_TYPE

|OPERATOR_ROLE
|- Monitor the transactions.
|USER_TYPE

|SERVICE_ROLE
|- Connect Service
|APPLICATION_TYPE
|===

NOTE: By default, super admin has an operator role as well.
