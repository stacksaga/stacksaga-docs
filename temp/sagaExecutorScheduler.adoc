= Customize Schedulers for SagaTemplate

by default stacksaga uses `Schedulers.boundedElastic()` for executing the transaction after handing over transaction to `SagaTemplate`.
If you want to customize the scheduler, you can do it by providing your own `Scheduler` bean in the application context with the name `sagaExecutorScheduler`.

For example:

[source,java]
----
//org.stacksaga.Stacksaga.BEAN_EVENT_LISTENER_SCHEDULER is a constant string "sagaExecutorScheduler"
@Bean(name = org.stacksaga.Stacksaga.BEAN_EVENT_LISTENER_SCHEDULER)
public Scheduler sagaExecutorScheduler() {
    return Schedulers.newBoundedElastic(10, 100, "saga-executor");
}
----
